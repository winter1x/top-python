Протестируйте функцию reverse() с длинным текстом. 
Эта функция принимает любую строку, возвращает новую перевернутую строку.

def reverse(string):
    if string is None:
        return None
    return string[::-1]

Cоздайте необходимые данные. 
У вас должны получиться как минимум два файла - один для входных данных (исходный текст) 
и файл с результатом (перевернутая строка).
